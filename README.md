# 🔁 Estruturas de Repetição em Java

## 📌 Descrição

Este repositório reúne uma série de exercícios desenvolvidos em Java com o objetivo de praticar estruturas de repetição (`for` e `while`), além do uso de condições e entrada de dados.

Os exercícios abordam diferentes situações práticas, como validação de dados, contagem, classificação e controle de fluxo.

---

## 🚀 Exercícios

### 🔹 Exercício 1 (FOR) – Números Ímpares

Programa que recebe um número inteiro e exibe todos os números ímpares até esse valor.

---

### 🔹 Exercício 2 (WHILE) – Validação de Senha

Sistema que solicita uma senha ao usuário e continua pedindo até que a senha correta (2002) seja informada.

---

### 🔹 Exercício 3 (FOR) – Intervalo de Valores

Programa que lê uma quantidade de números e verifica quantos estão dentro do intervalo [10, 20] e quantos estão fora.

---

### 🔹 Exercício 4 (WHILE) – Quadrantes

Programa que recebe coordenadas (X, Y) e informa em qual quadrante o ponto se encontra. O programa encerra quando um dos valores for zero.

---

### 🔹 Exercício 5 (WHILE) – Posto de Combustível

Sistema que registra a preferência de combustível dos clientes (álcool, gasolina ou diesel) e exibe a quantidade de cada tipo. O programa termina quando o valor 4 é informado.

---

## 🖥️ Tecnologias Utilizadas

* Java
* Programação estruturada
* Entrada de dados com `Scanner`

---

## 📚 Conceitos Aplicados

* Estruturas de repetição (`for`, `while`)
* Estruturas condicionais (`if/else`)
* Contadores e acumuladores
* Entrada e saída de dados
* Controle de fluxo
